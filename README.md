# Machine-Learning
## Overview
This repository contains various Machine Learning models and projects demonstrating concepts like supervised learning, unsupervised learning, clustering, classification, and regression. Each project includes:
- Dataset
- Data preprocessing
- Model building
- Evaluation metrics

## Technologies Used
- Python 3.13
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib & Seaborn
- Scikit-learn
